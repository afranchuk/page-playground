# MACROPAD code

This is the main sketch for the Page Playground, targeting the RP2040 on the Adafruit MACROPAD.

## Dependencies

This requires the following libraries to be installed (with dependencies) through the Arduino
Library Manager:
- Adafruit APDS9960 Library
- Adafruit BNO08x
- Adafruit NeoPixel
- Adafruit SH110X
- Adafruit seesaw Library
- Button (though the use of this could be removed pretty easily)
- MsgPack
- Packetizer
- [Quaternion](https://github.com/carrino/Quaternion): **not available through the library manager**
- RotaryEncoder
